Timoney - quick setup
1. Put this Timoney folder in your webserver root (eg. XAMPP htdocs).
2. Import timoney.sql into phpMyAdmin to create the database.
3. Update DB credentials in config.php if needed.
4. Start Apache + MySQL in XAMPP.
5. Open http://localhost/Timoney/public/index.html
6. Register, login, and use the dashboard.
7. For AI: run Ollama locally or change ai_ollama.php to point to your LLM.
